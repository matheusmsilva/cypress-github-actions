## Cypress - Github actions

Workflow of continuos testing with Cypress in Github Actions

## Stacks
- Cypress
- Javascript
- Cypress Dashboard
- Tesults

## Running

1. Install the dependencies
```
yarn / npm install
```

2. Run UI mode
```
yarn cypress open / npx cypress open 
```

3. Run headless mode
```
yarn cypress run / npx cypress run 
```
